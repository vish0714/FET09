- Make directory/folder
mkdir [folder-name]

- Delete directory/folder
rmdir [folder-name]

- Clear the terminal
clear

- Print/Present Working Directory
pwd

- List all files and folders
ls

- Change folder 
cd [folder-name]

- Go back one directory
cd ..

- Create files
touch [filename.ext]

- Delete a file
rm [filename.ext]

Make Multiple directory
&&
